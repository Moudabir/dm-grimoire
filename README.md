## 📖 What You Get

| Feature | Description |
|---|---|
| 📜 **Track Everything** | NPCs with actual personalities, encounters that make sense, loot you won't forget, and locations your players will return to — one dashboard, no hunting through five files |
| ⚡ **Auto-Save** | Stop paranoid-hitting Ctrl+S. Changes save instantly. Your 3 AM plot twists will survive a laptop death |
| 🌐 **Works Everywhere** | Laptop during the game, phone on a break, tablet at the table — Firebase keeps every device in sync |
| 🎛️ **The DM Cockpit** | Your personal mission control, built for running sessions — not generic spreadsheet nonsense |

## 🚀 Get Started

**You need:** Node.js 16+ and pnpm (or npm). That's it.

```bash
git clone https://github.com/Moudabir/dm-grimoire.git
cd dm-grimoire
pnpm install
pnpm dev
```

Open **`http://localhost:5173`** and start building your world.

## 🎲 How It Works in Practice

1. **Spin up** your campaign — give it a world
2. **Add** the people, places, and dangers that live there
3. **Drop in** session notes, plot hooks, random ideas — everything
4. **Load** the DM Cockpit when you run a session
5. **Everything auto-saves.** You focus on the story. That's it.
